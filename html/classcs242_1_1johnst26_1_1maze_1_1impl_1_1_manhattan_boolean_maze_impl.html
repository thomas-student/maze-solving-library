Detailed Description

An implementation of the Maze Interface. This is a Manhattan-Boolean implementation, in that the spaces of the maze are stored as an 2D array of booleans containing whether or `not the space is valid or invalid (whether it is a wall or not). The Manhattan component of the name signifying that the distance between points is based on the Manhattan distance formula.
